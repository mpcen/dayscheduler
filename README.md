# A Simple Day Schedule Viewer - Built With Love Using React
## To Install:
	- clone repository
	- cd into repository
	- npm install
	- npm start
	- navigate to http://localhost:3000/

### Browsers Tested:
	- Google Chrome 57.0.2987.133 (64-bit)
